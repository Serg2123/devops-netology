First Record

по добавленному в корне .gitignore ничего не будет игнорироваться, т.к. он пустой
будут применяться следующие правила игнорирования для файлов в поддиректории terraform (т.к. там .gitignore не пустой и со смыслом):
1. любые файлы в подпапке .terraform, которая сама м.б. внутри любого кол-ва вложенных подпапок, т.е. фактически исключается содержимое всех найденных подпапок .terraform
2. любые файлы с расширением .tfstate
3. любые файлы в названии которых есть ".tfstate"
4. конкретный файл crash.log
5. любые файлы с расширнием .tfvars
6. конкретный файл override.tf
7. конкретный файл override.tf.json
8. любые файлы в конце названия которых есть _override.tf
9. любые файлы в конце названия которых есть _override.tf.json
10. конкретный файл .terraformrc
11. конкретный файл terraform.rc
